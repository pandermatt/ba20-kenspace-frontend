## Developing

### Setup

```bash
git clone git@github.com:pandermatt/ba20-kenspace-frontend.git
cd ba20-kenspace-frontend
npm install
```

### Run
Compiles and hot-reloads for development
```bash
npm run serve
```

### Build
Compiles and minifies for production
```bash
npm run build
```

### Lints
Lints and fixes files
```bash
npm run lint
```
